# Task 04: KMP Frontend UI & State

Build the Compose Multiplatform UI with Decompose for navigation and state management.

## Goal
Establish a high-quality dashboard interface that can display chat sessions and run commands.

## Sub-tasks
- [x] Set up **Decompose** components for:
    - `RootComponent` (Main Navigator)
    - `DashboardComponent` (The main hub)
- [x] Implement the UI layout using Compose Multiplatform:
    - Sidebar for chat sessions and model selection.
    - Main panel for messages or the terminal.
- [x] Set up SQLDelight local caching to store sessions and messages on-device.
- [x] Implement UI for:
    - Basic Dashboard structure.
    - Chat Input.

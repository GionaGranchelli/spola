# Task 09: File Control

Implement file pull and push support.

## Goal
Let Spola Client transfer files through the backend in a safe, explicit way.

## Sub-tasks
- [ ] Define file transfer DTOs and API contracts.
- [ ] Implement backend pull/push endpoints.
- [ ] Add UI entry points for file actions.
- [ ] Show progress and error states.
- [ ] Record file actions for auditing.

## Test required
- Pull a file from host.
- Push a file to host.
- Confirm errors are reported cleanly.

## Acceptance criteria
- File operations are exposed in the UI.
- Backend supports file transfer endpoints.
- File actions are auditable and explicit.
